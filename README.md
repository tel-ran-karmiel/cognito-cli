# CLI Application for generating Cognito tokens (access, id and refresh tokens)
## Preconditions
- AWS credentials<br>
- Cognito user/s <br>
- Cognito application client without additional secret 
## Environment Variables
- LOGGER_LEVEL - logging level of pino <br>
- REGION - AWS Region <br>
- CLIENT_ID - client id of cognito application client
